# 카카오 알림톡 템플릿

> 카카오비즈센터에서 직접 등록 필요 (심사 1~3일)
> Solapi PFID + 템플릿 코드 연동

## CNEC_PROPOSAL_001 — 공구 초대

```
#{크리에이터명}님, 새로운 공구 초대가 도착했어요.

브랜드: #{브랜드명}
캠페인: #{캠페인명}
기간: #{캠페인기간}
커미션: #{커미션율}%

아래 링크에서 상세 내용을 확인해주세요.

[버튼] 초대 확인하기 → #{수락URL}

— 크넥샵
```

## CNEC_PROPOSAL_002 — 상품 추천

```
#{크리에이터명}님, 새로운 상품 추천 요청이 도착했어요.

브랜드: #{브랜드명}
상품: #{상품명}
커미션: #{커미션율}%

내 샵에 상시 추천 상품으로 등록하실지 확인해주세요.

[버튼] 상품 확인하기 → #{수락URL}

— 크넥샵
```

## 환경변수

| 변수명 | 설명 |
|--------|------|
| SOLAPI_API_KEY | Solapi API 키 |
| SOLAPI_API_SECRET | Solapi API 시크릿 |
| SOLAPI_SENDER_PHONE | 발신번호 (사전 등록 필수) |
| KAKAO_ALIMTALK_PFID | 카카오 플러스친구 ID |
