# 카카오 알림톡 템플릿 (팝빌)

> 팝빌 콘솔에서 템플릿 등록 + 카카오 심사 (1~3일)
> 팝빌 senderKey + templateCode 연동

## 환경변수

| 변수명 | 설명 |
|--------|------|
| POPBILL_LINK_ID | 팝빌 링크 ID |
| POPBILL_SECRET_KEY | 팝빌 시크릿 키 |
| POPBILL_CORP_NUM | 사업자등록번호 (하이픈 제외) |
| POPBILL_KAKAO_SENDER_KEY | 카카오 알림톡 발신프로필 키 |
| POPBILL_IS_TEST | 테스트 모드 (true/false) |

## 신규 등록 필요 템플릿

### CNECSHOP_013 공구 초대

```
#{크리에이터명}님, 새로운 공구 초대가 도착했어요.

브랜드: #{브랜드명}
캠페인: #{캠페인명}
커미션: #{커미션율}%

아래 링크에서 상세 내용을 확인해주세요.

[버튼] 초대 확인하기 → #{수락URL}

— 크넥샵
```

### CNECSHOP_014 상품 추천

```
#{크리에이터명}님, 새로운 상품 추천 요청이 도착했어요.

브랜드: #{브랜드명}
상품: #{상품명}
커미션: #{커미션율}%

내 샵에 상시 추천 상품으로 등록하실지 확인해주세요.

[버튼] 상품 확인하기 → #{수락URL}

— 크넥샵
```
