# 미처리 알림 트리거 (별도 검토 필요)

진단 리포트(docs/EMAIL_DIAGNOSIS_20260423.md)에서 발견된 항목 중,
sendNotification 호출 자체가 없어 별도 비즈니스 결정이 필요한 3건.

## 1. 브랜드 회원가입 환영 이메일

- **이벤트**: 브랜드가 회원가입 완료 시
- **수신자**: 브랜드 담당자
- **결정 필요 사항**: 가입 즉시 발송 vs 승인 후 발송, 이메일 내용

## 2. 구매자 회원가입 환영 이메일

- **이벤트**: 구매자가 회원가입 완료 시
- **수신자**: 구매자
- **결정 필요 사항**: 환영 이메일 내용, 쿠폰/혜택 포함 여부

## 3. 캠페인 생성 시 크리에이터 풀 알림

- **이벤트**: 새 캠페인이 생성(DRAFT)될 때
- **수신자**: 전체 활성 크리에이터 또는 타겟 크리에이터
- **결정 필요 사항**: DRAFT 시점 vs RECRUITING 시점(현재 RECRUITING 시작 알림은 구현 완료), 수신 대상 범위
